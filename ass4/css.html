About CSS

CSS stands for Cascading Style Sheets. It is used to style and design the layout of web pages.

CSS allows you to add colors, fonts, and spacing to your content.

CSS styles

  1. Selectors and Declarations:> CSS uses selectors to target elements and declarations to style them.
  2. Box Model: Consists of content, padding, border, and margin for element layout.
  3. Responsive Design:Adapts pages for different devices with media queries and frameworks.
