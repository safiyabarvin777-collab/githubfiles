About HTML

HTML stands for HyperText Markup Language. It is the standard language used to create web pages.

HTML provides structure to web content using elements and tags.

Basic HTML Tags.

  1. HTML:root element that defines html document.
  2. head:Contains meta-information about the document .
  3. title:Specifies the title of the web page, which appears on the browser tab.
  4. body: Contains the visible content of the webpage.
